# Tic Tac Toe Ruby

https://travis-ci.org/andrewMacmurray/tic-tac-toe-ruby.svg?branch=master

Command Line Tic Tac Toe in Ruby

## Get up and running

Install the dependencies with

```
> bundle install
```

Run the tests with

```
> bundle exec rspec
```

Run the application with

```
> bin/console
```
