# Tic Tac Toe Ruby

Command Line Tic Tac Toe in Ruby

## Get up and running

Install the dependencies with

```
> bundle install
```

Run the tests with

```
> bundle exec rspec
```

Or you can run the tests in the background with guard

```
> bundle exec guard
```

Guard will re-run the tests if any changes are saved in `lib` or `spec`
